# Movie app
Aplicativo que apresenta em forma de lista os resultados da base de dados yts-proxy.

## Tecnologias utilizadas
- React
  - hooks
  - styled components
  - styled reset
- Axios

## Como executar?

```bash
$ git clone https://github.com/EdsonLuiz/movie-app

$ cd movie-app

$ npm i

$ npm start
```
